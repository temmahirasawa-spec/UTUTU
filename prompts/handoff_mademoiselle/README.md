## 3. 動画について（実装上の注意）

新 {08} の右上セルは**動画**です。現状の型定義（`lib/projects.ts` の `ProjectBlock`）では `processStep.images` / `imageGrid.images` は `{ src, alt }` の**画像専用**で、動画を持てません。`Hero` 型のみ `video?: string` を持っています。

→ 以下のいずれかで対応してください（推奨は A）:

**A. メディアアイテムに任意 `video` を追加**
```ts
// 型を拡張
type Media = { src?: Img; video?: string; alt: string };
// processStep.images?: Media[]  に変更し、
// 描画側 ProjectStory.tsx で item.video があれば <video autoplay loop muted playsinline> を出す
```
{08} の4枠は `[ {画像}, {video:'…'}, {画像}, {画像} ]` の順（右上が動画）。

**B. {08} 専用の新ブロック型を追加**（`mediaGrid` 等）して2×2・16:9・動画対応を表現。

動画ファイルも `public/images/mademoiselle/`（ユーザー指定ディレクトリ）にあります。`<video>` は `autoplay loop muted playsinline` 想定（デザイン参照と同じ）。
