# UTUTU 実績詳細ページ 原稿 日本語化プロンプト（Claude Code 用）

実績詳細ページの構成（レイアウト・セクション構造・スクロール演出）は**今のままで変更しない**。変更するのは**原稿（コピー）だけ**。現在すべて英語で書かれているのを、**日本語主体**に書き換える。

## 大方針

1. **日本語を主体にする。** 見出し・本文・リード・説明はすべて日本語を主にする。英語は、デザイン上のアクセント（kicker、ラベル、セクション見出しの英字など既存トーン&マナーで英字を使っている箇所）にとどめ、**読ませる原稿は日本語**にする。
2. **トーン&マナーは既存サイト（UTUTU）を踏襲。** 日本語は Noto Sans JP、明朝は使わない。`word-break: keep-all` / `line-break: strict` を日本語見出しに適用。英主・日従の既存ルールが残る箇所（ヒーローの大見出しなど）は、英字タイトル＋日本語サブ、の構造を保ったまま日本語サブを充実させる。
3. **下記リサーチ済みの各ブランド情報を原稿に反映する。** コーポレートサイトの公式ストーリーを土台に、外部情報で得た事実（創業年・背景・商品特性・展開）で肉付けする。ただし**数値や固有情報は「※要確認」を付けて鵜呑みにさせない**（下記「注意」参照）。
4. レイアウト・画像配置・スクロール演出・コンポーネント構造は**一切触らない**。`lib/projects.ts`（または原稿データの定義箇所）の**テキスト値だけ**を日本語に差し替える。

## 各ブランドの原稿素材（これを使って日本語原稿を構成する）

> 使い方：各ブランドの「詳細ページ」の各セクション（ヒーロー / 概要 / ストーリー）に、以下の素材を割り当てて自然な日本語原稿にする。**そのままコピペせず、UTUTU のトーンで整える**。事実関係は下記に準拠。

### 共通の背景（全ブランド横断・ストーリーの軸に使える）

- 運営は **株式会社YORKYS ENTERTAINMENT**（神戸市中央区／代表取締役 板倉洋輔）。「食とデザインの力で想像を超えるエンターテインメントを」を掲げる。
- **原点は YORKYS BRUNCH（2014年、兵庫・夙川/西宮）。** サーフィン好きのオーナーがオーストラリアで体感した「ブランチ」という朝の食文化を持ち帰り、リコッタチーズを練り込んだスフレ系のふわふわパンケーキで関西を代表する人気店に。
- そのパンケーキ生地の**リコッタチーズ**に着目して生まれたのが **FROMA（チーズダイニング）**。さらにスイーツ領域へ展開して **YORKYS Creperie（スペシャリティ・クレープ）**、**PIECE OF BAKE（生ドーナツ・ボンボローニ）** が派生。**1つの原点から枝分かれした系譜**である、という物語が全ブランドに通底する。
- UTUTU（洋輔 × 天真）は、この YORKYS グループのブランド/クリエイティブを横断的に手掛けてきた。実績詳細ページは「UTUTU が何をデザインしたか」を語る視点で書く（店舗紹介ではなく、**クリエイティブ実績**として）。

### 01. YORKYS BRUNCH（ブランチ・カフェ）

- タグライン：**GOOD BRUNCH, GREAT DAY!**
- コア：「ブランチという豊かな時間を、日常のスタンダードに」。兵庫・夙川で2014年オープン、YORKYS の原点。
- ストーリー素材：オーナーがオーストラリア滞在中に感じた朝の時間の大切さ（日の出とともに海に入り、海からあがって食べる朝食）。「お客様にゆっくりくつろいでもらえる場所をつくりたい」という動機。シェフ・クリエイターと料理と空間を一緒に作る姿勢。「休日の朝、いつもより少し遅く起きて食べる、ちょっとだけリッチなブランチ」。
- 商品/特徴：リコッタチーズ入りのスフレ系ふわふわパンケーキ（看板）、エッグベネディクト、フレンチトースト。夜はワインも楽しめる業態。夙川カトリック教会を望むロケーション。
- UTUTU の関わり（実績として書く軸）：ブランド戦略・ロゴ/VI・店舗グラフィック・Web・モバイルオーダー（Orderly）等、原点ブランドの世界観づくり。※実際に担当した範囲に合わせて調整。

### 02. YORKYS Creperie（スペシャリティ・クレープ）

- タグライン：**Specialty Crepe / Crazy for crepe?**
- コア：「素材と製法を極めた、大人のためのスペシャリティ・クレープ」。ただのクレープではなく"ワンランク上"。
- ストーリー素材：「クレープが好きで好きでたまらない私たち」。美味しいクレープができるたびすぐ届けたくなる、お客様の幸せな笑顔を増やしたい、という作り手の想い。
- 商品/特徴：**北海道産クリーム**と**国産フルーツ**を贅沢に使用。**モチモチ食感**の生地（商品ごとに配合を変えて1つずつ丁寧に焼く）。季節の旬フルーツ。梅田・名古屋・吉祥寺・うめきた等を展開。梅田 阪急三番街で行列・SNSで話題に。2021年に名古屋初出店（イオンモール Nagoya Noritake Garden）。
- UTUTU の関わり：ロゴ/VI、店舗ファサード（名古屋店リニューアル等）、AIビジュアル、印刷物、FC展開のためのクリエイティブ。※担当範囲に合わせる。

### 03. PIECE OF BAKE（生ドーナツ・ボンボローニ）

- タグライン：**Bakery & Sweets Factory**
- コア：「出来立ての温もりを届ける、街の小さなファクトリー」。看板は生ドーナツ「ボンボローニ」。
- ストーリー素材：**ボンボローニ**はイタリア・トスカーナの伝統菓子。2022年（梅田）誕生。独自開発のふわふわ生地に、独自製法で"ふわじゅわ食感"。職人が丁寧に作る。小さな感動を届けるため日々新しい味を開発。
- 商品/特徴：**数種類の小麦粉をブレンド**したふわふわ生地＋**北海道産クリーム**の軽さ。**冷めても美味しい独自生地**。揚げたてにクリームを搾り入れる製法。フレーバー多数（プレーン/クレマ/ピスタチオ/キャラメル/チョコ/ラズベリー/レモン等＋限定）。梅田・名古屋・吉祥寺・マリンピア神戸 等を展開、FACTORY 併設。
- UTUTU の関わり：ロゴ/VI（手描き風ロゴ＋イエロー）、店舗グラフィック、商品ビジュアル、パッケージ等。※担当範囲に合わせる。

### 04. FROMA（チーズ・ダイニング）

- タグライン：**Cheese Dining**
- コア：「まるで地下の秘密のチーズ貯蔵庫。五感で味わうチーズ・ダイニング」。
- ストーリー素材：**YORKYS BRUNCH のパンケーキ生地に入っているリコッタチーズに着目**し、チーズを主役に据えて生まれた業態。神戸国際会館の20周年リニューアルに伴いオープン（神戸国際会館SOL B2F）。「地下の秘密のチーズ貯蔵庫」のような特別な空間というコンセプト。作りたてのフレッシュチーズをその場で味わえる**店内チーズ工房併設**。
- 商品/特徴：モッツァレラ・カマンベール・ゴルゴンゾーラに加え、ブッラータ・カチョカバロなどの高級チーズ。シカゴピザ、ボルケーノ/ボルチーニ系パスタ、フレッシュチーズのビュッフェ（ランチ）、独創的なチーズケーキ。着席・立食の大人数パーティ対応（公式記載に幅があるため**人数は「※要確認」**）。
- UTUTU の関わり：ブランド/VI、空間に紐づくグラフィック、メニュー/販促、Web 等。※担当範囲に合わせる。

## 原稿を書くときのルール

- **店舗紹介文の引き写しにしない。** UTUTU の実績ページなので、「私たちはこのブランドをどう捉え、何をデザインしたか」という制作者視点のストーリーに再構成する。ブランド公式の想いは"引用・要約"として織り込みつつ、UTUTU の語りにする。
- ヒーロー：英字タイトル（ブランド名）＋日本語の短いリード（そのブランドの核を1〜2行）。
- 概要：日本語で「制作領域・分野・担当・期間」等を整理。メタのラベルは既存の英字ラベル（Industry / Role / Deliverables / Timeframe 等）を残してよいが、**値は日本語**で。
- ストーリー：上の「ストーリー素材」を使い、緩急のあるブロック（フルスクリーン画像＋短い詩的な日本語、左右分割で説明文、余白の引用文）に流し込む。**4ブランドが金太郎飴にならないよう**、各ブランドの固有エピソード（BRUNCH＝豪州の朝/サーフィン、FROMA＝パンケーキのリコッタから派生、PIECE OF BAKE＝トスカーナのボンボローニ、Creperie＝北海道産クリーム×国産フルーツのモチモチ）を必ず核に据える。
- 日本語のトーンは、UTUTU の世界観（静かで芯がある、装飾と機能の両立、引き算）に合わせる。過剰な「！」やSNS的な軽さは避け、落ち着いた品のある文体。
- 既存の英語原稿は**置き換える**（英日併記で冗長にしない）。デザイン上どうしても英字が要る箇所のみ英字を残す。

## 注意（事実の扱い）

- 上記の年号・店舗数・人数・素材などは Web リサーチに基づくが、**時点により変化している可能性がある**。原稿に具体数値（席数・店舗数・創業年など）を入れる場合、データ側（`lib/projects.ts` のコメント等）に `// ※要確認:` を付け、天真さんが最終確認できるようにする。**不確かな数値を断定で本文に埋め込まない**（曖昧なら「関西発」「複数店舗を展開」等の表現に留める）。
- 担当範囲（UTUTU が実際に何を手掛けたか）は、天真さんしか正確に分からない。**Role / Deliverables は仮置きにし、`// TODO: 実際の担当範囲に修正` を残す**。
- 画像・レイアウトは触らない。原稿テキストのみ変更。
